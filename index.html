Prompt / Instructions for Scouters

Welcome to the REBUILT 2026 Pit Scouting App!

Please follow these instructions carefully to ensure accurate and complete data collection:

Scouter Info

  • Enter your name and the team number you are scouting.
  • Make sure there is no profanity or inappropriate language.

Robot Info

  • Drive Type: Select only one drive option.
  • Intake / Auto Source: If “Both” is selected, do not select other options.
  • Capacity & Preload: Preload cannot exceed the robot’s capacity.

Autonomous (Auto)

  • Enter the number of preloaded balls and their source.
  • Make sure “Both” selection rules are followed here as well.
  • Only valid numbers are accepted; warnings will appear for incorrect entries.

Teleop

  • Enter climb level, position, and notes about robot performance.
  • Avoid using any offensive language or racial slurs in notes.

Saving Data

  • Press Save & QR to generate a QR code.
  • If any warnings appear, fix the issues before saving.

Reset

  • Use Reset All to clear the form and start fresh.

⚠️ Warnings will appear if:

  • Multiple drives are selected
  • “Both” is selected with other options
  • Preload exceeds capacity
  • Notes or scouter name contain profanity or racial slurs

This app works offline, so your data is stored locally until scanned by a Wi-Fi device.
